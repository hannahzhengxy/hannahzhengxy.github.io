Java之String

发表于 2022-05-06 分类于 Java

Java的八个基本数据类型分别是int, short, long, byte, boolean, char, float, double。 其余都是引用类型，如数组和String。 引用类型指向的是一个对象，不是原始值。

在Java里，参数传递都是按值传递，按值传递是传递值的拷贝，而引用传递是传递引用的地址值。

在这个例子中，word1等于word2，而word3不等于word1。因为word1和word2指向同一个对象，而word3创建了一个新的对象。
