Java installation on Windows

发表于 2022-05-06 分类于 Java

Java runs on Java virtual machine(JVM), thus we need to install the Java Development Kit(JDK).

  1. Go to Oracle website to download the newest version of JDK. For Windows, select Windows with x64 Installer.

  2. After installing the JDK, we need to add the directory to the path of the environment variable.

  • Go to the file where you installed the JDK, and go to the bin file, copy the directory path of the current folder.
  • Go to the control panel, select System and Security, select System, select Advanced System Settings.
  • In the advanced system settings, select Environment Variables, choose path and edit, choose new and paste the directory path of the bin file of JDK.
  1. Check if Java is installed successfully:
    In the CMD, check the Java version: If it shows the version you just installed, then Congratulations!
